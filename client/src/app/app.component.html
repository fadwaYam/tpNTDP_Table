Demonstration de l'affichage du tableau

Description

Il s'agit d'un exemple d'affichage d'une table des restaurants en utilisation angular 4 et le module cors pour autoriser l'accès de l'applicatio du client à l'API server, les données seront acheminés du serveur sous format json vers le client, en utilisant le typescript ainsi on aura la possibilité d'interprété ces données pour enfin les affichés sous format d'une table html5

la recherche se fait par nom des restaurants et ce à l'aide d'un pipe que j'ai installé monsieur
